# ReaMD - Markdown Viewer for REAPER

Dockable markdown viewer for audio production workflows.
Link text fragments to timeline items, use teleprompter mode for VO,
and format text with AI.

## Key Features

- **Scenario Linking** - Connect headings and table rows to timeline items
- **Cue List** - Sorted overview of all linked fragments, click to jump
- **Teleprompter** - Auto-scrolling VO display
- **AI Parse** - Convert messy text to clean markdown (Claude Haiku/Sonnet/Opus)
- **Full Markdown** - Headers, lists, tables (with proper text wrapping),
  code blocks, strikethrough
- **Search** - Ctrl+F in-document find with next/prev
- **time:// links** - Markdown links can jump REAPER's edit cursor
- **Cross-platform** - Windows, macOS, Linux

## Requirements

- REAPER 7.0+
- ReaImGui 0.10+
